# Учебное приложение: статическая и динамическая типизация

**ВКР · Ляпин Дмитрий Александрович · ПГНИУ ИКНТ · 2026**

Приложение на Python 3.10+ с графическим интерфейсом (tkinter) для изучения
статической и динамической типизации.

## Быстрый старт

```bash
pip install pygments mypy
python main.py
```

## Структура проекта

```
typing_app/
├── main.py                  # точка входа
├── requirements.txt
├── core/
│   ├── __init__.py
│   ├── type_checker.py      # интеграция с mypy, AST-анализ
│   ├── examples.py          # 5 демонстрационных сценариев
│   └── exercises.py         # 6 учебных упражнений
├── ui/
│   ├── __init__.py
│   ├── main_window.py       # главное окно (ttk.Notebook)
│   ├── editor.py            # редактор кода + подсветка Pygments
│   ├── results_panel.py     # панель диагностических сообщений
│   └── exercise_panel.py    # интерактивный учебный модуль
└── utils/
    ├── __init__.py
    └── helpers.py           # вспомогательные утилиты
```

## Функциональность

| Вкладка | Что делает |
|---------|-----------|
| **Примеры** | 5 тем × 3 варианта кода (динамический / статический / с ошибкой). Запуск кода и проверка mypy. |
| **Упражнения** | 6 заданий нарастающей сложности. Двухэтапная проверка: AST-анализ + mypy. Прогрессивные подсказки. |
| **О приложении** | Архитектура, инструкция, сведения об авторе. |

## Требования

- Python ≥ 3.10
- pygments ≥ 2.15
- mypy ≥ 1.5
